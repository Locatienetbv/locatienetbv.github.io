Welcome to the Locatienet API Documentation

Welcome to the official documentation for the Locatienet API. Here you'll find detailed references, usage examples, and guides to help you integrate Locatienet services into your applications.

Getting Started

Our API allows you to:

  • Geocode addresses and locations across the Netherlands.
  • Calculate routes and distances for multiple locations.
  • Optimize routing based on vehicle type, time, and other constraints.
  • Access detailed information about locations, streets, and administrative areas.
